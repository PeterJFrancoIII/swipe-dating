# Approvals

Production release requires authentic, dated, named human approvals. Autonomous agents must not author approval signatures.

Required before production (see Phase 20):

| Artifact | Role | Freshness |
|---|---|---|
| Legal counsel sign-off | Legal | ≤90 days |
| Privacy/DPIA acceptance | Privacy | ≤90 days |
| External security review | Security | ≤180 days |
| Trust & safety staffing | T&S lead | ≤30 days |
| Executive launch approval | Executive | ≤30 days |
| App-store compliance matrix | Mobile lead | ≤30 days |
| Infra/production account attestation | Infra | ≤7 days |

Place signed artifacts in this directory only via protected human workflow. Filenames should include role, date (UTC), and document hash.
