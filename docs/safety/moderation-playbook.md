# Moderation playbook (DRAFT)

**Updated:** 2026-07-20

## Principles

- Safety features free and always available.
- Prefer user block + report; escalate to human review for P0/P1.
- Agents must not adjudicate real user cases or send real enforcement notices.

## Severity (illustrative)

| Level | Examples | Target |
|---|---|---|
| P0 | CSAM, imminent violence, active NCII campaign | Immediate human path |
| P1 | Credible threats, stalking, nonconsensual sharing | Same-day human |
| P2 | Harassment, spam, scams | Queue SLA `CHANGE_ME` |
| P3 | Policy edge cases | Batch review |

## Appeals

Documented channel `CHANGE_ME`; no automated permanent bans without human policy for P0/P1.
