# Trafficking indicators playbook (human-owned stub)

**Status: UNAPPROVED / HUMAN-OWNED**

## Triggers

- User report: trafficking / exploitation
- Pattern signals from human review (not automated ML on messages in MVP)

## Operator steps (draft)

1. Preserve evidence in vault; minimize re-victimization (limit access).
2. Escalate immediately to T&S lead and legal.
3. Follow jurisdictional reporting obligations — **human legal counsel only**.
4. Document actions in case system without PII in tickets.

## Autonomous agent boundary

Do not simulate law-enforcement filings or fabricate hotline numbers in production configs.
